# Roadmap

Planned skills to add to the catalog:

- **pagination** — page through large result sets safely (cursor vs. offset, stable ordering, limits)
- **search** — add full-text or filtered search with relevant ranking and pagination
- **notifications** — send transactional email/push reliably with templating and opt-out
- **idempotency** — make write endpoints safe to retry with idempotency keys
- **audit-log** — record who did what, when, for security and compliance

Contributions welcome — open an issue or PR to suggest a skill.
