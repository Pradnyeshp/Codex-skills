# Roadmap

Planned skills to add to the catalog:

- **file-uploads** — accept uploads safely (type/size limits, virus scan, off-server storage, signed URLs)
- **connection-pooling** — size and manage DB/HTTP connection pools to avoid exhaustion
- **retry-backoff** — a general client-side retry/backoff/circuit-breaker playbook for outbound calls

Recently shipped: `pagination`, `idempotency`, `audit-log`, `search`, `notifications`.

Contributions welcome — open an issue or PR to suggest a skill.
